# Overview

The `@loaders.gl/csv` module handles tabular data stored in CSV and TSV format 
[CSV/DSV file format](https://en.wikipedia.org/wiki/Comma-separated_values). 

## Installation

```bash
npm install @loaders.gl/core @loaders.gl/csv
```
