# Gaussian Splat PLY Fixtures

`train-1000.ply` is a 1000-splat binary subset of the Apache-2.0 `Voxel51/gaussian_splatting` dataset on Hugging Face.

Source file:
https://huggingface.co/datasets/Voxel51/gaussian_splatting/resolve/main/FO_dataset/train/point_cloud/iteration_7000/point_cloud.ply

The fixture was generated by rewriting the PLY header from `element vertex 741883` to `element vertex 1000` and copying the first 1000 binary vertex records unchanged. It is intended for parser/schema coverage; use the full sample in `deck.gl-data/formats/ply/gaussian-splat/` for visual examples.
