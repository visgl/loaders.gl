# Format Logos

This folder centralizes the logos intended for the docs home page and format landing pages.

- `format-logos.json` is the source of truth that maps each `Formats` sidebar entry to one shared asset in this directory.
- `local-copy` entries were copied from existing repo assets so the home page does not need to depend on per-module image paths.
- `upstream-download` entries were downloaded from official upstream sources.
- `generated-fallback` entries are lightweight in-repo badges for formats where we did not find a stable canonical upstream logo during this pass.

## Upstream Downloads

- `maplibre-logo.svg`: [MapLibre brand assets](https://maplibre.org/brand-assets/)
- `mapbox-logo.png`: [Mapbox brand guidelines](https://www.mapbox.com/about/press/brand-guidelines)
- `khronos-logo.png`: [Khronos logo usage guidelines](https://www.khronos.org/legal/logo-usage-guidelines)
- `w3c-xml-logo.svg`: [W3C XML icon](https://www.w3.org/Icons/XML/xml.svg)
- `arcgis-logo.jpg`: [Esri media relations brand assets](https://www.esri.com/en-us/about/media-relations/b-roll)
