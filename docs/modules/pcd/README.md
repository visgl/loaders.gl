# Overview

The `@loaders.gl/pcd` module handles the the [Point Cloud Data](/docs/modules/pcd/formats/pcd), which stores 3D point cloud data). 

## Installation

```bash
npm install @loaders.gl/pcd
npm install @loaders.gl/core
```
