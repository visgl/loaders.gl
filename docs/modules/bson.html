Overview

The @loaders.gl/bson module provides support for the BSON format. The BSON format stores arbitrary (loosely structured) data largely equivalent to the textual JSON format.

Installation​

npm install @loaders.gl/core @loaders.gl/bson

Loaders and Writers​

Loader
BSONLoader
BSONWriter

Attribution​

This module is a wrapper around MongoDB js-bson module, which is under Apache 2.0 license.
