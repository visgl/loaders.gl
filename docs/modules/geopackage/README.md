# Overview

![ogc-logo](../../images/logos/ogc-logo-60.png)

The `@loaders.gl/geopackage` module handles the OGC [GeoPackage](https://www.geopackage.org/) format.

## Installation

```bash
npm install @loaders.gl/geopackage
```
